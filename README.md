# 🌱 Predicting Plant Growth Stages Using Environmental and Management Data

## 📊 Project Overview

This project focuses on predicting and analyzing plant growth stages using environmental and agricultural management data through an interactive Power BI dashboard.

The main objective is to help farmers, greenhouse managers, agricultural researchers, and agritech professionals make data-driven decisions for improving crop productivity and sustainability.

By analyzing environmental conditions and farming practices, the dashboard provides meaningful insights into the factors that influence plant growth and development.

---

# 🎯 Objectives

- Visualize environmental and management factors affecting plant growth
- Identify optimal conditions for healthy plant development
- Build an interactive and user-friendly Power BI dashboard
- Apply predictive analytics using Python integration in Power BI
- Support sustainable and efficient agricultural practices

---

# 📂 Dataset Features

The dataset includes several important agricultural and environmental parameters:

| Feature | Description |
|---|---|
| **Soil Type** | Type of soil used for cultivation |
| **Sunlight Hours** | Daily exposure to sunlight |
| **Water Frequency** | Watering schedule and frequency |
| **Fertilizer Type** | Type of fertilizer applied |
| **Temperature** | Environmental temperature conditions |
| **Humidity** | Atmospheric humidity levels |
| **Growth Milestones** | Plant growth stage indicators |

---

# 🛠️ Tools & Technologies Used

- **Power BI** — Data visualization and dashboard development
- **CSV Dataset** — Agricultural data source

# 📈 Dashboard Features

The Power BI dashboard provides:

- Interactive filtering and slicers
- Environmental condition analysis
- Growth stage prediction insights
- Correlation analysis between farming practices and plant growth
- Trend visualization for temperature, humidity, and sunlight
- Comparative analysis of fertilizer and watering methods

---

# 🔍 Key Insights

- Identifies environmental conditions that promote healthy growth
- Highlights relationships between soil, water, and fertilizer usage
- Helps optimize agricultural resource management
- Supports early-stage decision-making for crop improvement
